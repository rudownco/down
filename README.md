# Down

A Swift macOS app built with SwiftUI.

## Requirements

- macOS 14.0 or later
- Xcode 15+ (or Swift 5.9+)

## Getting Started

### Build

```bash
swift build
```

### Run

```bash
swift run Down
```

Or run the built executable directly:

```bash
.build/debug/Down
```

### Open in Xcode

You can open the package in Xcode for a full development experience:

```bash
open Package.swift
```
